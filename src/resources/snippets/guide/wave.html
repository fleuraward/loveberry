How to change the colour of the wave for the wave text

  1. Copy the image link, https://files.catbox.moe/db8ne8.png and open Photopea. With this image link, click File → Open More → Open from URL. Paste your link into the box and press OK
  3. From here, you should see this screen (picture 1) create a new layer (click or you can click layer → new → layer) and fill it with the colour you want the wave to be.
  6. Right click on your full coloured in layer and click clipping mask! Now you should have a recoloured wave.
  8. click file → export as → png to save it! then upload it to your site and replace the image link within the code to your new image!
